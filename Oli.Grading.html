    send_score_to_lms(section, user, resource_access, access_token_provider)

    View Source

    If grade passback services 2.0 is enabled, sends the current state of a ResourceAccess score for the current user to the LMS.

    If sent successfully, returns {:ok, :synced}

    If grade passback not enabled, returns {:ok, :not_synced}

    If error encountered, returns {:error, error}
