    find_objective_attachments(resource_id, publication_id)

    View Source

    For a given objective resource id and a given project's publication id, this function will find all pages and activities that have the objective attached to it.

    This function will return an activity more than once if that activity contains multiple parts with the objective attached to it.

    The return value is a list of maps of the following format: %{ id: the revision id resource_id: the resource id title: the title of the resource slug: the slug of the revision part: the part name, or "attached" if pertaining to a page }
